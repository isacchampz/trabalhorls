# Trabalho Power BI - Bolsa Família

## Descrição

Projeto desenvolvido no Power BI utilizando dados do programa Bolsa Família.

O objetivo foi realizar análises por Unidade Federativa (UF) a partir dos dados disponibilizados, criando visualizações para auxiliar na interpretação das informações.

## Análises Realizadas

### 1. Soma de NIS Favorecido por UF

Visualização que apresenta a soma dos registros de NIS Favorecido agrupados por Unidade Federativa, permitindo identificar a distribuição dos beneficiários entre os estados.

### 2. Soma de Competência por UF

Visualização que apresenta a soma dos valores de competência agrupados por Unidade Federativa, possibilitando a análise dos períodos processados em cada estado.

### 3. Soma do Valor da Parcela por UF

Visualização que apresenta a soma dos valores das parcelas agrupados por Unidade Federativa, permitindo comparar os montantes distribuídos entre os estados.

## Ferramentas Utilizadas

* Power BI Desktop
* Power Query
* Arquivo CSV como fonte de dados

## Estrutura do Projeto

O projeto foi desenvolvido utilizando o formato Power BI Project (.pbip), permitindo versionamento dos arquivos através do GitHub.
